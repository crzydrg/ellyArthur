# ELLYARTHUR DISC Assessment
> 엘리아더헤어 채용 면접용 성향 분석 검사 시스템

**v1.0** · HTML 단일 파일 · 외부 서버 불필요

---

## 개요

지원자가 태블릿으로 직접 응시하고, 면접관이 코드로 잠긴 결과지를 확인하는 구조의 DISC 성향 검사 앱입니다. 헤어 디자이너·스탭 채용에 특화된 현장 상황 문항과, 성향·신뢰도·문화핏에 따라 자동 생성되는 면접 질문이 포함되어 있습니다.

---

## 사용 방법

### 응시 플로우

```
① 지원자 이름 · 포지션 입력
        ↓
② 응답 방법 안내 화면
        ↓
③ 40문항 응시 (문항당 가까운 것 + 먼 것 각 1개 선택)
        ↓
④ "작성 완료 · 면접관에게 기기를 넘겨주세요" 화면
        ↓
⑤ 면접관 코드 입력 (4자리)
        ↓
⑥ 결과 확인 · 이미지 저장 · 카카오톡 공유
```

### 결과 화면 구성

| 섹션 | 내용 |
|------|------|
| I. PRIMARY TYPE & PROFILE | 주 유형 + DISC 4개 점수 바 |
| II. JOB FIT ANALYSIS | 직무 적합도 % (원형 차트) + 강점/주의점 |
| III. RELIABILITY & CULTURE FIT | 신뢰도(일관성) + 문화 적합도 |
| IV. INTERVIEW QUESTIONS | 성향·신뢰도·문화핏 기반 맞춤 면접 질문 |

## 주요 기능

### 변별력 설계

- **강제선택형 (Forced-Choice)** — 문항당 가장 가까운 것 + 가장 먼 것을 각각 1개씩 선택. "좋게 보이려는" 답변 원천 차단
- **선택지 순서 무작위 섞기** — 검사 시작 시 40문항 전체 선택지 순서를 랜덤 생성·고정. 매 검사마다 다른 순서로 표시되어 패턴 유추 불가
- **일관성 체크 (Lie Detector)** — 유사 문항 6쌍 교차 검증. 33% 이하 시 면접 질문에 심층 검증 항목 자동 추가

### 문항 구성 (총 40문항)

| 구분 | 문항 수 | 내용 |
|------|---------|------|
| 일반 성향 | 10 | 의사결정, 스트레스, 팀 역할 등 |
| 미용실 현장 상황 | 15 | 진상 고객, 인턴 실수, 피크타임, 원장 지시 등 |
| 가치관·동기 | 7 | 일하는 이유, 5년 후, 좋은 리더상 등 |
| 일관성 체크 | 8 | 앞선 문항과 유사한 내용으로 재질문 |

### 면접 질문 자동 생성

매 검사마다 질문 풀에서 무작위로 선택됩니다. 총 질문 풀은 50개입니다.

| 카테고리 | 풀 크기 | 선택 수 |
|----------|---------|---------|
| 주 유형 (D/I/S/C) | 각 6개 | 3개 |
| 보조 유형 | 각 6개 | 1개 |
| 포지션 (디자이너/스탭) | 각 4개 | 2개 |
| 신뢰도 기반 (3단계) | 각 2~3개 | 1~2개 |
| 문화핏 기반 (3단계) | 각 2~3개 | 1~2개 |

**신뢰도·문화핏 점수에 따라 질문 강도가 자동 조정됩니다.**

- 신뢰도 낮음 (0~33%) → 포장 의심 심층 질문 2개
- 문화핏 낮음 (0~54%) → 팀워크·가치관 검증 질문 2개
- 모두 높음 → 성장 심화 질문으로 대체

### 이미지 저장 / 공유

| 기능 | 내용 |
|------|------|
| 전체 저장 (FULL) | 성향 분석 + 면접 질문 포함 |
| 요약 저장 (SUMMARY) | 성향 결과만 (질문지 제외) |
| 카카오톡 공유 | 요약 이미지를 공유 시트로 바로 전송 |

저장 파일명 자동 생성 예시: `ELLYARTHUR_DISC_홍길동_20260422_FULL.png`

## 기술 스펙

- **단일 HTML 파일** — 외부 서버, 데이터베이스, 로그인 불필요
- **외부 라이브러리 없음** — 인터넷 연결 없이도 전체 기능 작동 (폰트 제외)
- **Canvas API로 인포그래픽 직접 렌더링** — html2canvas 불사용, 저장 품질 안정적
- **Web Share API** — iOS/안드로이드 공유 시트 연동으로 카카오톡 직접 전달
- **시스템 폰트 사용** — Apple SD Gothic Neo, Georgia 등 기기 내장 폰트

### 실행 환경

| 환경 | 방법 |
|------|------|
| **iPad / iPhone** | Safari로 열기 → 홈 화면에 추가 권장 |
| **안드로이드** | Chrome으로 열기 |
| **PC** | 모든 브라우저 |

### 문항 추가

`QUESTIONS` 배열에 아래 형식으로 추가합니다.

```javascript
{
  t: "문항 내용 (나는 ~할 때)",
  o: [
    { txt: "D 성향 선택지", type: "D" },
    { txt: "I 성향 선택지", type: "I" },
    { txt: "S 성향 선택지", type: "S" },
    { txt: "C 성향 선택지", type: "C" }
  ]
}
```

> **주의**: 선택지 순서는 D→I→S→C로 작성하되, 화면 표시 시 자동으로 무작위 섞임

### 면접 질문 추가

```javascript
// 유형별 질문 풀 (INTERVIEW_QUESTIONS)

  { q: "질문 내용", intent: "면접관 확인 포인트" },
  ...
]

// 포지션별 질문 (POSITION_QUESTIONS)

  { q: "질문 내용", intent: "면접관 확인 포인트" },
  ...
```
